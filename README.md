# prosemirror-test
A temporary package until ProseMirror is added to npm


**How to use**

Use prosemirror.js in your project. Everything is available as `pm.`  . For example `Pos` is `pm.Pos`.

**How to build**

Run `./build.sh` 

**How to build again after updates to ProseMirror**

Run `./rebuild.sh`


**Build requirements**

`python-virtualenv`
